# REST APIs using Node.js, MongoDB, Fastify and Swagger

> A Node.js project

## Build Setup

``` bash
> install dependencies
npm install

> serve will be live on [http://127.0.0.1:3000](http://127.0.0.1:3000)
cd src
node server.js

> sswagger ui on
[http://127.0.0.1:3000/documentation](http://127.0.0.1:3000/documentation)
```
## Prerequisites
> Nodejs
> MongoDB
